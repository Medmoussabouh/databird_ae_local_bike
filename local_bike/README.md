Welcome to your new dbt project!

### Model Layers
| Layer | Folder | Purpose |
|--------|---------|----------|
| **staging** | `models/staging/` | Cleans and standardizes source data |
| **intermediate** | `models/intermediate/` | Joins and enriches multiple datasets |
| **marts** | `models/marts/` | Business-ready models for analytics and dashboards |

### Using the starter project

Try running the following commands:
- dbt run
- dbt test
